Chat Home Screen — Design Options

The Tab5 needs a proper chat experience. Not just a single conversation — a home that shows history, lets you pick sessions, and feels like a real messaging app.

What's missing from the current Chat?

Has Now
  • Single conversation
  • Text input + mic button
  • Mode badge (tappable to cycle)
  • New Chat button (clears history)
  • Live status (Processing/Ready/Speaking)
  • Thinking + tool indicators
  • Streaming response bubbles
Missing
  • Session history list
  • Previous conversations
  • Session metadata (date, preview, mode)
  • Model selector for Cloud mode
  • Session search/filter
  • Delete old sessions
  • Token/cost info per session
  • Persistent messages across close/reopen

Pick the architecture

How should session history integrate with the Chat screen?

A

Drawer / Sidebar

Swipe from left edge to reveal a session list panel (like ChatGPT mobile). Chat stays visible behind it. Tap a session to load it. Minimal screen real estate used.

B

Chat Home → Conversation (Recommended)

Tapping "Chat" from nav shows a Chat Home with session cards (like WhatsApp/iMessage). Tap a card to enter that conversation. "New Chat" at top. Each card shows: last message preview, timestamp, message count, mode used. Back returns to Chat Home, not to the main home screen.

C

Dropdown Selector

Keep current single-screen but add a dropdown at top showing recent sessions. Minimal change. But limited space for metadata and doesn't feel like a proper messaging app.

What else should the Chat Home show?

+

Model Picker

Dropdown to select LLM model before starting a chat. Shows available models from OpenRouter + local Ollama models.

+

Quick Actions Bar

"Search the web", "Set a timer", "Remember something" — one-tap tool shortcuts above the session list.

+

Memory Preview

Small section showing what Tinker remembers about you (3 recent facts). Tap to see all.

+

Voice Activity Visualizer

Small waveform or orb animation showing when Tinker is listening/speaking. Makes the device feel alive even when idle.
