Chat Home Screen — Design Options

The Tab5 needs a proper chat experience. Not just a single conversation — a home that shows history, lets you pick sessions, and feels like a real messaging app.

What's missing from the current Chat?

Has Now
  • Single conversation
  • Text input + mic button
  • Mode badge (tappable)
  • New Chat button (clears)
  • Status indicator (Processing/Ready)
  • Thinking/tool indicators
Missing
  • Session history list
  • Previous conversations
  • Session metadata (date, preview)
  • Model selector for Cloud mode
  • Session search/filter
  • Delete old sessions
  • Token/cost info per session

Pick the architecture

How should session history integrate with the Chat screen?

A

Drawer / Sidebar

Swipe from left edge to reveal a session list panel (like ChatGPT mobile). Chat stays visible behind it. Tap a session to load it. Current conversation stays active until you switch.

B

Chat Home → Conversation (Recommended)

Tapping "Chat" from nav shows a Chat Home with session cards (like WhatsApp). Tap a card to enter that conversation. "New Chat" button at top. Each card shows: last message preview, timestamp, message count, mode used. Back from conversation returns to Chat Home.

C

Dropdown Selector

Keep current single-screen layout but add a dropdown at the top showing recent sessions. Minimal change, least disruption. But limited space for session metadata.
